Задание 5

Создание ЦВЗ с помощью модуляции мощности фрагментов.

Реализовать внедрение ЦВЗ в аудиофайл путем модуляции мощности амплитуд.
Сформировать для ЦВЗ некоторую строку из симоволов (например, сообщение означающее авторство данного аудиофайла).
С помощью библиотек bitstring или bitarray или средствами Python перевести строку из символов ЦВЗ в битовую
последовательность. Внедрение битов ЦВЗ осуществить с помощью сплайна B2i.
На основе указанного сплайна построить функции u_ и u+,
которые будет осуществлять модуляцию в зависиомсти от значения бита.
Для внедрения последовательности битов ЦВЗ оригинальный сигнал нужно разбить на
неперсекающиеся последовательные фрагменты одинаковой длины (~10мс).
В каждый фрагмент внедряется свой бит. Внедрение осуществляется путем умножения амплитуд на функцию u_ или u+.
Извлечь внедренное сообщение можно путем сравнения мощности фрагментов оригинального файла и файла с ЦВЗ.
Для некоторой строки необходимо продемонстрировать внедрение и извлечение ЦВЗ.

Проверить метод на устойчивость к атакм:
- добавление случайного шума небольшой амплитуды
- конвертация аудиофайла по схеме: wav -> mp3 -> wav.
